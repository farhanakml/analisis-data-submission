# Bike Sharing Dataset - Submission Analisis Data Dicoding
Submission Tugas Proyek Akhir Belajar Analisis Data Dicoding berupa pengeksploresian data dan pembuatan dashboard.

## Cara Mengakses :
1. Clone this repository
2. Install the necessary packages via cmd or terminal by running `pip install -r requirements.txt`
3. Run the code `streamlit run dashboard.py`
