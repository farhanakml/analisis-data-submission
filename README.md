# Bike Sharing Dataset - Final Project Submission
Final Project Submission for Belajar Analisis Data on Dicoding in the form of data exploration and creating dashboard.

You can access the dashboard [here!](https://data-analysis-farhanakml.streamlit.app).

## How to Access :
1. Clone this repository
2. Install the necessary packages via cmd or terminal by running `pip install -r requirements.txt`
3. Run the code by running `streamlit run dashboard.py`
